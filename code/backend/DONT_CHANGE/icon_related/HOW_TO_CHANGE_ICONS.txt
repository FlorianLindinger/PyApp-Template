=======================================
==== HOW TO CORRECTLY MAKE AN ICON ====
=======================================

TLDR:
Replace icon.png in code/icons/ with the image you want and run:

    code/icons/regenerate icons.lnk

You can also replace these overlay PNGs in code/icons/ if you want custom sub-icons:

    settings.png
    stop.png
    log.png
    success.png
    failure.png
    crash.png

The icon generator creates these .ico files in code/icons/:

    icon.ico
        Icon version of icon.png.

    settings.ico
        Icon version of icon.png with settings.png overlaid in the bottom-right corner.

    stop.ico
        Icon version of icon.png with stop.png overlaid in the bottom-right corner.

    log.ico
        Icon version of icon.png with log.png overlaid in the bottom-right corner.

    success.ico
        Icon version of icon.png with success.png overlaid in the bottom-right corner.

    failure.ico
        Icon version of icon.png with failure.png overlaid in the bottom-right corner.

    crash.ico
        Icon version of icon.png with crash.png overlaid in the bottom-right corner.

The shortcut generator uses those .ico files for the generated program shortcuts.

=======================================
==== INPUT FILES ======================
=======================================

User-editable input images:

    code/icons/icon.png
    code/icons/settings.png
    code/icons/stop.png
    code/icons/log.png
    code/icons/success.png
    code/icons/failure.png
    code/icons/crash.png

Keep the file names unchanged.

If an input PNG is missing, the generator uses the matching fallback image from:

    code/DONT_CHANGE/icon_related/default_icon.png
    code/DONT_CHANGE/icon_related/default_settings.png
    code/DONT_CHANGE/icon_related/default_stop.png
    code/DONT_CHANGE/icon_related/default_log.png
    code/DONT_CHANGE/icon_related/default_success.png
    code/DONT_CHANGE/icon_related/default_failure.png
    code/DONT_CHANGE/icon_related/default_crash.png

If a user image is still exactly the template fallback image, the generator also treats it as fallback.

=======================================
==== GENERATION =======================
=======================================

Run this shortcut:

    code/icons/regenerate icons.lnk

That shortcut currently launches:

    code/DONT_CHANGE/backend_python/python.exe code/DONT_CHANGE/icon_related/generate_icons.py

The generator script is:

    code/DONT_CHANGE/icon_related/generate_icons.py

It uses Windows PowerShell/WPF rendering to resize images cleanly, preserve aspect ratio,
add transparent padding, and write a multi-resolution .ico file with these sizes:

    256, 128, 64, 48, 32, 16

The optional first command-line argument is the overlay scale factor.
Default:

    0.6

Manual example:

    code/DONT_CHANGE/backend_python/python.exe code/DONT_CHANGE/icon_related/generate_icons.py 0.6

Icons should be generated this way instead of with a quick online converter, because Windows
icons need multiple embedded sizes to scale well in shortcuts, taskbar entries, and Explorer.

=======================================
==== PILLOW / PIL ALTERNATIVE =========
=======================================

The current generator above does not need Pillow. If you prefer a shorter script that is easier
to edit manually, this Pillow version has virtually identical behavior:

    - reads icon.png and overlay PNGs from code/icons/
    - falls back to the default images when user images are missing or unchanged from the template
    - writes icon.ico and overlay .ico files to code/icons/
    - keeps aspect ratio, transparent padding, and the same icon sizes
    - uses the same bottom-right overlay behavior and overlay scale factor

This version requires Pillow:

    pip install pillow

Alternative script:

import os
import sys
import zlib
from pathlib import Path

from PIL import Image


ICON_SIZES = (256, 128, 64, 48, 32, 16)
DEFAULT_OVERLAY_SCALE_FACTOR = 0.6

SCRIPT_DIR = Path(__file__).resolve().parent
USER_ICON_DIR = (SCRIPT_DIR / "../../icons").resolve()
FALLBACK_ICON_DIR = SCRIPT_DIR
OUTPUT_DIR = USER_ICON_DIR

FALLBACK_IMAGE_IDS = {
    "base": "200x200:82f0d3c0",
    "settings": "200x200:71db6cbb",
    "stop": "200x200:83248fd0",
}


def image_id(path: Path) -> str:
    image = Image.open(path).convert("RGBA")
    rgba_bytes = image.tobytes()
    crc = zlib.crc32(rgba_bytes) & 0xFFFFFFFF
    return f"{image.width}x{image.height}:{crc:08x}"


def pick_icon_path(user_path: Path, fallback_path: Path, fallback_image_id: str, label: str) -> Path:
    if user_path.exists():
        if image_id(user_path) == fallback_image_id:
            print(f"Using fallback {label} icon.")
            return fallback_path
        return user_path

    print(f"Using fallback {label} icon because {user_path.name} is missing.")
    return fallback_path


def square_layer(image: Image.Image, size: int, background_color=(0, 0, 0, 0)) -> Image.Image:
    src = image.convert("RGBA")
    scale = size / max(src.width, src.height)
    new_width = max(1, round(src.width * scale))
    new_height = max(1, round(src.height * scale))

    resized = src.resize((new_width, new_height), Image.Resampling.LANCZOS)
    canvas = Image.new("RGBA", (size, size), background_color)
    offset_x = (size - new_width) // 2
    offset_y = (size - new_height) // 2
    canvas.paste(resized, (offset_x, offset_y), resized)
    return canvas


def save_ico(layers: list[Image.Image], output_path: Path) -> None:
    layers[0].save(
        output_path,
        format="ICO",
        sizes=[(size, size) for size in ICON_SIZES],
        append_images=layers[1:],
    )


def create_icon(image_path: Path, output_path: Path) -> None:
    image = Image.open(image_path).convert("RGBA")
    layers = [square_layer(image, size) for size in ICON_SIZES]
    save_ico(layers, output_path)


def create_composite_icon(
    base_path: Path,
    overlay_path: Path,
    output_path: Path,
    overlay_scale_factor: float = DEFAULT_OVERLAY_SCALE_FACTOR,
) -> None:
    base = Image.open(base_path).convert("RGBA")
    overlay = Image.open(overlay_path).convert("RGBA")

    scale = max(base.width, base.height) / max(overlay.width, overlay.height) * overlay_scale_factor
    overlay_width = max(1, round(overlay.width * scale))
    overlay_height = max(1, round(overlay.height * scale))
    overlay = overlay.resize((overlay_width, overlay_height), Image.Resampling.LANCZOS)

    composite = base.copy()
    pos_x = base.width - overlay_width
    pos_y = base.height - overlay_height
    composite.paste(overlay, (pos_x, pos_y), overlay)

    layers = [square_layer(composite, size) for size in ICON_SIZES]
    save_ico(layers, output_path)


if __name__ == "__main__":
    if len(sys.argv) > 1:
        overlay_scale_factor = float(sys.argv[1])
    else:
        overlay_scale_factor = DEFAULT_OVERLAY_SCALE_FACTOR

    base_icon_path = pick_icon_path(
        USER_ICON_DIR / "icon.png",
        FALLBACK_ICON_DIR / "default_icon.png",
        FALLBACK_IMAGE_IDS["base"],
        "base",
    )
    settings_icon_path = pick_icon_path(
        USER_ICON_DIR / "settings.png",
        FALLBACK_ICON_DIR / "default_settings.png",
        FALLBACK_IMAGE_IDS["settings"],
        "settings",
    )
    stop_icon_path = pick_icon_path(
        USER_ICON_DIR / "stop.png",
        FALLBACK_ICON_DIR / "default_stop.png",
        FALLBACK_IMAGE_IDS["stop"],
        "stop",
    )

    try:
        create_icon(base_icon_path, OUTPUT_DIR / "icon.ico")
        print("Generated: icon.ico")
    except Exception as error:
        print(f"Error creating icon.ico: {error}")

    try:
        create_composite_icon(
            base_icon_path,
            settings_icon_path,
            OUTPUT_DIR / "settings.ico",
            overlay_scale_factor,
        )
        print("Generated: settings.ico")
    except Exception as error:
        print(f"Error creating settings.ico: {error}")

    try:
        create_composite_icon(
            base_icon_path,
            stop_icon_path,
            OUTPUT_DIR / "stop.ico",
            overlay_scale_factor,
        )
        print("Generated: stop.ico")
    except Exception as error:
        print(f"Error creating stop.ico: {error}")
