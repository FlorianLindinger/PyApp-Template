==========================
==== ASCII Font Files ====
==========================

These FIGlet/TOIlet font files were copied from the font data used by the TAAG Text to ASCII Art Generator:

https://patorjk.com/software/taag/

========================
==== License Status ====
========================

This folder contains fonts from different authors with different license statements. Treat the files individually; there is no single license that covers all of them.

| File | TAAG font name | Header license/status |
| --- | --- | --- |
| `standard.flf` | Standard | Header grants permission to modify the font if the modifier is named on a comment line. It does not include a complete modern license text. Keep attribution and verify if strict redistribution terms matter. |
| `coder_mini.flf` | Coder Mini | Header says the font is free to use and distribute under the MIT License. |
| `future.tlf` | Future | Header says WTFPL version 2, with no warranty. |
| `future_smooth.tlf` | Future Smooth | Header says WTFPL version 2, with no warranty. The header also credits the Future Smooth modifications. |
