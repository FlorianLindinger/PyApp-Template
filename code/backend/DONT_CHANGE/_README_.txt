==================================
=== Folder content explanation ===
==================================

"backend" refers in the total of this framework more or less as everything except the main_script.py. But within the this folder it is more meant as a distinction between what is run on the embedded (included) Python ("backend") and the Python that gets installed to run main_script.py ("frontend").

- B: Short name folder with short name batch files as short shortcut targets (batch files because of antivirus false flag safety and shorter paths in shortcut target. Also possibility to install backend python on run. Shortness because of Windows path length limits.)
- backend_packages
- backend_python: embedded python modified as described bx scripts/setup/finish_backend_installation.py
- backend_test_tools
- future: Stuff for future PyApp-Template development
- icon_related
- scripts
- _README_.txt: see _README_.txt
- backend_packages_list.txt
- PyApp-Template_TODO.txt
- PyApp-Template_VERSION_0.8a.txt
