################################
HOW TO CORRECTLY MAKE AN ICON
################################

TLDR: Replace icon.png/settings.png/stop.png with images that you want (keep name) and run generate_icons.lnk

################################

Icons have to be generated propery to nicely scale with mulitple sizes in a .ico file:

Easiest way is to run generate_icons.lnk in this folder which auto installs the ImageMagick program if not installed already. For that the images icon.png/settings.png/stop.png have to be in this folder and can be changed to get different output icons.

Alternatively you can install ImageMagick manually (https://imagemagick.org/script/download.php) manually run the commands (see below) that generate_icons.lnk runs. 
If you want to have just a png->ico for settings.ico or stop.ico (instead of the main icon.ico with settings/stop signs in the corner) you can use and change the names in the first command below.
If you want a different size of the corner signs for settings.ico/stop.ico, you can change the "0.6" below.

magick "icon.png" -define icon:auto-resize=16,32,48,64,128,256 -compress zip "icon.ico"

magick icon.ico[5] -set option:BW "%[w]" -set option:BH "%[h]" ^
  ( settings.png -resize "%[fx:int(option:BW*0.6)]x%[fx:int(option:BH*0.6)]" ) ^
  -gravity southeast -composite ^
  -background none -alpha on -define icon:auto-resize=256,128,64,48,32,16 settings.ico
  
 magick icon.ico[5] -set option:BW "%[w]" -set option:BH "%[h]" ^
  ( stop.png -resize "%[fx:int(option:BW*0.6)]x%[fx:int(option:BH*0.6)]" ) ^
  -gravity southeast -composite ^
  -background none -alpha on -define icon:auto-resize=256,128,64,48,32,16 stop.ico
